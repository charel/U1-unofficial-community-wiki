# Feeder creaking or making noises, but feeding fine
## Description
When filament is passing through the feeder, one or both filament paths creak.
## Cause
Filament gets rolled through the feeder, but the rollers inside are plastic on plastic without any lubrification. 
## Solution
[Dissassemble the feeder and grease/lubricate the rollers.](../../guides/feeder-dissassemble.md)
