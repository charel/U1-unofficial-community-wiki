# U1 unofficial Community Wiki
The goal of this wiki is to document various pieces of information that are otherwise not well documented [on Discord](https://discord.gg/r8r46jSs).

It is not limited to Discord and can include links to other resources. 

## Contributing
You can contribute by submitting a PR. More information on contributing can be found in the [contribute.md page](https://charel.github.io/U1-unofficial-community-wiki/contribute). 

Please attribute the original author and link to the initial source, where possible. 

Don't overthink your contributions too much. "Good enough" is good enough. Better badly documented than not at all.
